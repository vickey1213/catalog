# Dynamic Multi-Category Catalog

A small React + Vite application that renders a responsive product catalog grouped by category and provides item detail navigation.

## Features
- Home screen with category sections for Cars, Bikes, Phones, and Computers
- Preview cards showing item images and key attributes
- Dedicated item detail page with dynamic rendering of all category-specific properties
- Mobile-friendly responsive layout
- Navigation using React Router

## Setup
1. Run `npm install`
2. Run `npm run dev`
3. Open the local URL shown in the terminal

## Files
- `src/App.jsx` — main UI and routing
- `src/data.json` — provided catalog data
- `src/index.css` — responsive styles
- `src/main.jsx` — React entry point
